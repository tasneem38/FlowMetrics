# 🚀 FlowMetrics – Webflow Showcase Website

FlowMetrics is a fully responsive SaaS-style product landing page built to demonstrate clean UI structure, layout hierarchy, CMS-style blog architecture, and modern frontend best practices.

This project is designed as a **Webflow Developer Internship showcase**, highlighting:

- Structured landing page sections
- Scroll-based animations
- Responsive layout
- CMS-style blog system
- Clean typography and spacing
- Consistent design system

### Blog System (CMS-Style)
The blog is powered by a JavaScript-based data source:

[posts.js](file:///c:/Users/sh29t/Desktop/Projects/FlowMwtrics/posts.js)

Each post includes:
- id
- title
- excerpt
- author
- date
- tags (array)
- cover image
- rich HTML content

This mimics Webflow CMS dynamic collections.

- Consistent color system
